Parcela subdividida

Eric Gorgens

1/17/2021

No delineamento em parcelas subdivididas existem dois tipos de tratamento: o principal e o secundrio. As parcelas so subdivididas no espao, ou no tempo. Depois que os tratamentos principais so sorteados nas parcelas, sorteia-se o tratamento secundrio nas subparcelas de cada parcela.

Indica-se o uso de parcelas subdivididas quando:

  1. A parcela uma unidade que pode receber vrios tratamentos secundrios. No setor florestal esta unidade pode ser uma vaso, ou mesmo uma rvore.
  2. No possvel instalar o experimento no esquema fatorial.
  3. O tratamento principal exige parcelas custosas, seja do ponto de vista financeiro ou do esforo.
  4. A busca pela preciso est no tratamento secundrio.
  5. Deseja-se que a Variao entre subparcelas seja menor que entre as parcelas.

O modelo estatstico do delineamento fatorial duplo inteiramente casualizado :

\[Y = Xmed + BLOCO + TRAT A + Erro Parcela + TRAT B + (TRAT A * TRAT B) + Erro Subparcela\]

O caso balanceado

Como exemplo de um desenho de parcela subdividida balanceado, vamos analisar um experimento de regenerao natural do sub bosque de trs formaes florestais. Nas parcelas de cada uma das rea naturais estudadas, implantaram-se subparcelas correspondendo a trs alturas de desrama. O efeito dos tratamentos foi medido atravs do nmero de indivduos regenerantes.

  • Fator nas subparcelas: Desrama 2, 5 e 7 metros

Analisando os grficos acima, os fatores parecem significativos, assim como a interao. Os dados apresentam-se consistentes, sem a presena pontos fora da tendncia. Infelizmente o pacote ExpDes.pt no inclui as anlises de pressuposio para o delineamento de parcelas subdivididas. Existem vrias maneiras de contornar esta questo, uma delas combinar as qualidades de diferentes pacotes. Aqui por exemplo, podemos rodar a ANOVA a partir de ambos os pacotes easyanova e ExpDes.pt. Assim podemos utilizar os testes estatsticos das pressuposies de um e a ANOVA e desdobramentos do outro.

require(easyanova)
## Loading required package: easyanova
## Warning: package 'easyanova' was built under R version 4.0.3
r.aov = ea2(sub, design=4)

A sintaxe da funo para anlise de parcela subdividida no pacote ExpDes.pt :

psub2.dic(fator1, fator2, repet, resp, quali = c(TRUE, TRUE),
          mcomp = "tukey", fac.names = c("F1", "F2"), 
          sigT = 0.05, sigF = 0.05)

Os dados por se tratarem de experimento desbalanceados com efeito da interao relevante devem ser analisados com o pacote easyanova. A funo ser ea2() e possui a sintaxe bsica:

ea2(data, design = 1, alpha = 0.05, cov = 4, list = FALSE, 
    p.adjust=1, plot=2)

Os dados devem ser apresentados na ordem:

  1. Fator nas parcelas
  2. Repetio
  3. Fator nas subparcelas
  4. Varivel resposta

O parmetro design deve ser ajustado para 4, resultando no seguinte comando:

require(easyanova)
r.aov = ea2(sub2, design=4)
